# Blackjack in Python

A command-line Blackjack card game created in Python.

## Features

- Standard 52-card deck
- Dealer AI
- Hit and Stand
- Ace value adjustment
- Score calculation
- Play multiple rounds

## Skills Demonstrated

- Python
- Object-Oriented Programming
- Git
- GitHub
- Problem Solving

## How to Run

```bash
python main.py
```
