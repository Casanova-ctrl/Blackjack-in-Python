# Blackjack in Python

A command-line implementation of the classic Blackjack card game built with Python. This project demonstrates object-oriented programming principles, clean code organization, and user interaction through a menu-driven interface.

---

## Project Overview

This application allows a player to compete against a computer-controlled dealer using the standard rules of Blackjack.

The project was designed to strengthen Python programming skills while demonstrating software design concepts such as classes, inheritance, encapsulation, modular programming, and user input validation.

## How to Run

Clone the repository:

```bash
git clone https://github.com/Casanova-ctrl/blackjack-in-python.git
```

Move into the project directory:

```bash
cd blackjack-in-python
```

Run the application:

```bash
python main.py
```

or

```bash
python3 main.py
```
